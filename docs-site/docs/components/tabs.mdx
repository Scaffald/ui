# Tabs

Tabbed navigation component with multiple visual variants. Supports horizontal and vertical orientations.

## Import

```tsx
import { Tabs } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Tabs defaultValue="tab1">
    <Tabs.Item value="tab1">
      <Tabs.Trigger><Text>Tab 1</Text></Tabs.Trigger>
      <Tabs.Content><Text>Content for Tab 1</Text></Tabs.Content>
    </Tabs.Item>
    <Tabs.Item value="tab2">
      <Tabs.Trigger><Text>Tab 2</Text></Tabs.Trigger>
      <Tabs.Content><Text>Content for Tab 2</Text></Tabs.Content>
    </Tabs.Item>
    <Tabs.Item value="tab3">
      <Tabs.Trigger><Text>Tab 3</Text></Tabs.Trigger>
      <Tabs.Content><Text>Content for Tab 3</Text></Tabs.Content>
    </Tabs.Item>
  </Tabs>
</ThemeProvider>`} />
