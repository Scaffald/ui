# Breadcrumbs

Hierarchical navigation breadcrumbs for showing the user's current location within a site structure.

## Import

```tsx
import { Breadcrumb } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Breadcrumb items={[{label: 'Home', href: '#'}, {label: 'Products', href: '#'}, {label: 'Details'}]} />
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/Breadcrumb/Breadcrumb.tsx) for the complete props API.

- Keyboard navigation support

- ARIA attributes included
