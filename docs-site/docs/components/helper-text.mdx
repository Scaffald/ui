# HelperText

Displays supplementary text below form fields to guide user input.

## Import

```tsx
import { HelperText } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={12} style={{maxWidth: 400}}>
    <HelperText>Passwords must be at least 8 characters</HelperText>
  </Stack>
</ThemeProvider>`} />
