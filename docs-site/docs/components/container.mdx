# Container

Responsive container with max-width constraints for centering and containing page content.

## Import

```tsx
import { Box, Text } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Box style={{maxWidth: 600, padding: 16, backgroundColor: '#f0f0f0'}}>
    <Text>Container content</Text>
  </Box>
</ThemeProvider>`} />
