# Row

Horizontal layout component with configurable spacing between children.

## Import

```tsx
import { Row } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Row gap={12}>
    <Text>Left</Text>
    <Text>Center</Text>
    <Text>Right</Text>
  </Row>
</ThemeProvider>`} />
