# Tooltip

Contextual help text that appears on hover or focus. Accessible and customizable.

## Import

```tsx
import { Tooltip } from '@scaffald/ui';
```

  <Tooltip content="This is a tooltip" visible={true} arrowPosition="down-center">
    <Button variant="outline" color="primary">Hover me</Button>
  </Tooltip>
</ThemeProvider>`} />
