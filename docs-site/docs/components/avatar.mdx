# Avatar

User profile image component with fallback initials. Supports different sizes and shapes.

## Import

```tsx
import { Avatar } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Row gap={12} style={{alignItems: 'center'}}>
    <Avatar initials="JD" size={48} color="primary" />
    <Avatar initials="AB" size={48} color="success" />
    <Avatar initials="XY" size={48} color="error" />
  </Row>
</ThemeProvider>`} />
