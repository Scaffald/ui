# Toggle

On/off switch component. Alternative to checkbox for boolean states.

## Import

```tsx
import { Toggle } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Toggle label="Enable notifications" />
</ThemeProvider>`} />
