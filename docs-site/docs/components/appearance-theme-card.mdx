# AppearanceThemeCard

Theme selection card for switching between light and dark appearance modes.

## Import

```tsx
import { AppearanceThemeCard } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Row gap={12}>
    <AppearanceThemeCard variant="light" />
    <AppearanceThemeCard variant="dark" />
  </Row>
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/AppearanceThemeCard/AppearanceThemeCard.tsx) for the complete props API.

### Common Props

All AppearanceThemeCard components accept standard React Native [`View` props](https://reactnative.dev/docs/view#props).
