# FileUpload

Drag-and-drop file upload component with progress tracking and file type validation.

## Import

```tsx
import { FileUpload, FileUploadDropZone, FileUploadList } from '@scaffald/ui';
```

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/FileUpload/FileUpload.tsx) for the complete props API.

## Sub-components

- `FileUploadDropZone` - Drag-and-drop target area
- `FileUploadList` - List of uploaded files
- `FileUploadItem` - Individual file item
- `FileUploadProgress` - Upload progress indicator

## Accessibility

- Keyboard accessible drop zone
- Screen reader announcements for upload status
