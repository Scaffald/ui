# Select

Dropdown selection component for choosing from a list of options.

## Import

```tsx
import { ResponsiveSelect } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <ResponsiveSelect
    options={[
      { label: 'Option 1', value: '1' },
      { label: 'Option 2', value: '2' }
    ]}
    placeholder="Select..."
  />
</ThemeProvider>`} />
