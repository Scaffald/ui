# Toast

Toast component from Scaffald UI component library.

## Import

```tsx
import { Toast } from '@scaffald/ui';
```

## Basic Usage

> Toast notifications are imperative and difficult to demo in a static preview. The examples below use Alert components to illustrate the visual styles.

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={12}>
    <Alert variant="success">Toast: Operation completed!</Alert>
    <Alert variant="error">Toast: Something went wrong</Alert>
  </Stack>
</ThemeProvider>`} />
