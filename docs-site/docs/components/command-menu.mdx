# CommandMenu

A searchable command palette for quick navigation and action execution. Includes CommandMenuItem for individual actions and CommandShortcut for keyboard shortcut badges.

## Import

```tsx
import { CommandMenu, CommandMenuItem, CommandShortcut } from '@scaffald/ui';
```

## Usage

CommandMenu renders as a modal overlay triggered by a keyboard shortcut (Ctrl+K or Cmd+K).

```tsx
const [isOpen, setIsOpen] = useState(false);

<CommandMenu
  open={isOpen}
  onClose={() => setIsOpen(false)}
  placeholder="Type a command or search..."
>
  <CommandMenuItem label="Go to Dashboard" icon="home" />
  <CommandMenuItem label="Settings" icon="settings" />
</CommandMenu>
```
