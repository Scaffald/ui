# ButtonGroup

A group of related buttons displayed together with shared styling and layout.

## Import

```tsx
import { ButtonGroup } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <ButtonGroup items={[{label: 'Left'}, {label: 'Center'}, {label: 'Right'}]} />
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/ButtonGroup/ButtonGroup.tsx) for the complete props API.

### Common Props

All ButtonGroup components accept standard React Native [`View` props](https://reactnative.dev/docs/view#props).
