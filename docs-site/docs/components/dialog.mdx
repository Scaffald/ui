# Dialog

Confirmation and alert dialogs for prompting user decisions or displaying important messages.

:::note
Dialog is an alias for the Modal component. All Modal props and behavior apply to Dialog as well.
:::

## Import

```tsx
import { Modal } from '@scaffald/ui';
```
