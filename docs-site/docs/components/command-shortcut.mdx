# CommandShortcut

Displays keyboard shortcut badges, commonly used within command menus and tooltips.

## Import

```tsx
import { CommandShortcut } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <CommandShortcut keys={['Ctrl', 'K']} />
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/CommandShortcut/CommandShortcut.tsx) for the complete props API.

### Common Props

All CommandShortcut components accept standard React Native [`View` props](https://reactnative.dev/docs/view#props).
