# FieldError

Displays validation error messages for form fields.

## Import

```tsx
import { FieldError } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={12} style={{maxWidth: 400}}>
    <FieldError message="This field is required" />
    <FieldError message="Please enter a valid email address" />
  </Stack>
</ThemeProvider>`} />
