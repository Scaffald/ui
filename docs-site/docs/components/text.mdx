# Text

Base text component with support for size, weight, and color variants.

## Import

```tsx
import { Text } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={8}>
    <Text size="lg" weight="bold">Large Bold</Text>
    <Text size="md">Medium Regular</Text>
    <Text size="sm" color="secondary">Small Secondary</Text>
  </Stack>
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/Text/Text.tsx) for the complete props API.

- Keyboard navigation support

- ARIA attributes included
