# CommandMenuItem

An individual item within a CommandMenu, representing a selectable action or navigation target.

## Import

```tsx
import { CommandMenuItem } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">

    <CommandMenuItem label="Go to Dashboard" />
    <CommandMenuItem label="Settings" />
    <CommandMenuItem label="Help & Support" />
  </Stack>
</ThemeProvider>`} />

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/CommandMenuItem/CommandMenuItem.tsx) for the complete props API.

## Examples

```tsx
// With icons and shortcuts
<CommandMenuItem label="Copy" icon="copy" shortcut="Ctrl+C" />
<CommandMenuItem label="Paste" icon="clipboard" shortcut="Ctrl+V" />
<CommandMenuItem label="Delete" icon="trash" destructive />
```
