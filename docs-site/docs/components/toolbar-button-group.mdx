# ToolbarButtonGroup

Groups multiple ToolbarButtons in a row, optionally wrapped in a glass pill container. Supports 1-7 buttons.

## Import

```tsx
import { ToolbarButtonGroup } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/ToolbarButtonGroup/ToolbarButtonGroup.types.ts) for the complete props API.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `buttons` | `ToolbarButtonConfig[]` | required | Button configurations (1-7) |
| `position` | `'top' \| 'bottom'` | `'bottom'` | Affects sizing |
| `glass` | `boolean` | `false` | Wrap in glass pill container |

## Accessibility

- Each button is individually focusable
- Group does not add its own semantic role
