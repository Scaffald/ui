# AppStoreButton

App store download button component supporting Apple App Store and Google Play Store badges.

## Import

```tsx
import { AppStoreButton } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <AppStoreButton store="apple" />
</ThemeProvider>`} />
