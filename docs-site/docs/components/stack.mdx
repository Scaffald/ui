# Stack

Vertical layout component with configurable spacing between children.

## Import

```tsx
import { Stack } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={12}>
    <Text>First</Text>
    <Text>Second</Text>
    <Text>Third</Text>
  </Stack>
</ThemeProvider>`} />
