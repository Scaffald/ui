# DatePicker

Calendar-based date selector with support for single date, date range, and time selection.

## Import

```tsx
import { DatePicker } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Text>DatePicker renders a calendar-based date selector with month navigation, day grid, and optional time input. See props documentation for configuration.</Text>
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/DatePicker/DatePicker.tsx) for the complete props API.

### Common Props

All DatePicker components accept standard React Native [`View` props](https://reactnative.dev/docs/view#props).

## Examples

```tsx
// Single date selection
<DatePicker
  value={selectedDate}
  onChange={(date) => setSelectedDate(date)}
  placeholder="Select a date"
/>

// Date range selection
<DatePicker
  mode="range"
  startDate={startDate}
  endDate={endDate}
  onChange={({ start, end }) => {
    setStartDate(start);
    setEndDate(end);
  }}
/>
```
