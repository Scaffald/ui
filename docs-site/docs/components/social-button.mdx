# SocialButton

Pre-styled buttons for social login providers (Google, Apple, etc).

## Import

```tsx
import { SocialButton } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={12} style={{maxWidth: 350}}>
    <SocialButton brand="google">Continue with Google</SocialButton>
    <SocialButton brand="apple">Continue with Apple</SocialButton>
  </Stack>
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/SocialButton/SocialButton.tsx) for the complete props API.

### Common Props

- `brand` - Social provider brand
- `children` - Button label text

## Accessibility

- Keyboard navigation support
- Screen reader compatible
