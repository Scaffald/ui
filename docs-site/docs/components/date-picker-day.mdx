# DatePickerDay

Individual day cell component used within the DatePicker calendar grid. Handles selection state, today highlight, and disabled states.

## Import

```tsx
import { DatePickerDay } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Text>DatePickerDay renders an individual day cell within the DatePicker calendar grid. It handles selection, today highlighting, and disabled states. Use the higher-level DatePicker for most use cases.</Text>
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/DatePickerDay/DatePickerDay.tsx) for the complete props API.

### Common Props

All DatePickerDay components accept standard React Native [`View` props](https://reactnative.dev/docs/view#props).

## Examples

```tsx
// Individual day cells (typically rendered by DatePickerBase)
<Row gap={4}>
  <DatePickerDay day={14} />
  <DatePickerDay day={15} selected />
  <DatePickerDay day={16} today />
  <DatePickerDay day={17} disabled />
</Row>
```
