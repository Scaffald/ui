# Heading

Semantic heading levels h1-h6 for structuring content hierarchy.

## Import

```tsx
import { Heading, H1, H2, H3 } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={8}>
    <H1>Heading 1</H1>
    <H2>Heading 2</H2>
    <H3>Heading 3</H3>
  </Stack>
</ThemeProvider>`} />
