# CommandMenuFooter

Footer section for the CommandMenu, typically displaying keyboard shortcuts or helpful hints.

## Import

```tsx
import { CommandMenuFooter } from '@scaffald/ui';
```

## Basic Usage

    <CommandMenuFooter />
  </Stack>
</ThemeProvider>`} />

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/CommandMenuFooter/CommandMenuFooter.tsx) for the complete props API.

All CommandMenuFooter components accept standard React Native [`View` props](https://reactnative.dev/docs/view#props).

## Examples

```tsx
// Inside a CommandMenu
<CommandMenu open={isOpen} onClose={handleClose}>
  <CommandMenuItem label="Dashboard" />
  <CommandMenuItem label="Settings" />
  <CommandMenuFooter />
</CommandMenu>
```
