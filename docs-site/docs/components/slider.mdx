# Slider

Range input component for numeric values. Supports steps and custom formatting.

## Import

```tsx
import { Slider, RangeSlider } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={16} style={{maxWidth: 300}}>
    <Slider value={50} />
    <RangeSlider value={[25, 75]} />
  </Stack>
</ThemeProvider>`} />
