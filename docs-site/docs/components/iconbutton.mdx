# IconButton

Icon-only button variant for compact actions like toolbars, close buttons, and navigation controls.

:::note
IconButton is achieved using the Button component with icon props and compact styling.
:::

## Import

```tsx
import { Button, Text } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Button variant="ghost" color="primary" style={{width: 40, height: 40, padding: 0}}>
    <Text>+</Text>
  </Button>
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/Button/Button.tsx) for the complete props API.

- Keyboard navigation support
- Screen reader compatible
- ARIA attributes included
