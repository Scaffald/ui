# DatePickerBase

Base wrapper component for DatePicker internals. Provides the calendar grid layout and shared context for day cells and header.

## Import

```tsx
import { DatePickerBase } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Text>DatePickerBase is an internal building block for the DatePicker component. It provides the calendar grid layout and context. Use the higher-level DatePicker component for most use cases.</Text>
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/DatePickerBase/DatePickerBase.tsx) for the complete props API.

### Common Props

All DatePickerBase components accept standard React Native [`View` props](https://reactnative.dev/docs/view#props).

## Examples

```tsx
// Typically used internally by DatePicker
<DatePickerBase
  month={currentMonth}
  year={currentYear}
  onDayPress={handleDayPress}
>
  <DatePickerHeader />
  {/* Day cells rendered here */}
</DatePickerBase>
```
