# Card

Container component for grouping related content. Supports headers, footers, and various visual styles.

## Import

```tsx
import { Card } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Card variant="elevated" padding="lg" style={{maxWidth: 350}}>
    <CardHeader title="Card Title" subtitle="Card subtitle" />
    <CardContent>
      <Text>This is the card content area. You can put any content here.</Text>
    </CardContent>
    <CardFooter>
      <Button variant="filled" color="primary" size="sm">Action</Button>
    </CardFooter>
  </Card>
</ThemeProvider>`} />
