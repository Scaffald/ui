# Spacer

Flexible spacing component for adding consistent gaps between elements.

## Import

```tsx
import { Spacer } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack>
    <Text>Above</Text>
    <Spacer size={24} />
    <Text>Below (24px gap)</Text>
  </Stack>
</ThemeProvider>`} />
