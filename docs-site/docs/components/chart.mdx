# Chart

SVG-based charting components for data visualization. Includes bar charts, donut charts, sparklines, progress rings, and more.

## Import

```tsx
import { BarChart, DonutChart, SparkLine, ProgressRing, LinearChart } from '@scaffald/ui';
```

## BarChart

Renders vertical bars from an array of numeric values.

<LivePreview code={`<ThemeProvider initialTheme="light">
  <BarChart data={[30, 55, 80, 45, 70, 90, 60, 40, 75, 50, 85, 65]} height={160} width={400} />
</ThemeProvider>`} />

## DonutChart

Circular chart for displaying proportional data.

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Row gap={32} style={{flexWrap: 'wrap', alignItems: 'center'}}>
    <DonutChart size="lg" data={[{ label: 'Engineering', value: 45 }, { label: 'Design', value: 25 }, { label: 'Marketing', value: 20 }, { label: 'Sales', value: 10 }]} />
    <DonutChart size="md" colorScheme="colorful" data={[{ label: 'Complete', value: 72 }, { label: 'Remaining', value: 28 }]} />
  </Row>
</ThemeProvider>`} />

## SparkLine

Compact inline trend indicator.

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Stack gap={12}>
    <Row gap={12} align="center">
      <Text size="sm" style={{width: 80}}>Revenue</Text>
      <SparkLine data={[10, 15, 12, 18, 22, 25, 30]} width={100} height={28} />
    </Row>
    <Row gap={12} align="center">
      <Text size="sm" style={{width: 80}}>Churn</Text>
      <SparkLine data={[8, 7, 9, 12, 15, 14, 18]} trend="down" width={100} height={28} />
    </Row>
  </Stack>
</ThemeProvider>`} />

## ProgressRing

Circular progress indicator with percentage label.

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Row gap={24} style={{flexWrap: 'wrap'}}>
    <ProgressRing value={25} label="Storage" size="sm" />
    <ProgressRing value={68} label="Tasks" size="md" />
    <ProgressRing value={92} label="Goals" size="lg" />
  </Row>
</ThemeProvider>`} />

## All Chart Components

| Component | Description |
|-----------|-------------|
| `BarChart` | Vertical bar chart |
| `StackedBarChart` | Stacked vertical bars |
| `LinearChart` | Line/area chart |
| `DonutChart` | Donut/ring chart |
| `CircleChart` | Full circle chart |
| `HalfPieChart` | Half-circle gauge |
| `SparkLine` | Compact inline trend |
| `ProgressRing` | Circular progress |
| `RadarChart` | Radar/spider chart |
