# Chart

SVG-based charting component for data visualization. Supports line, bar, area, and pie charts.

## Import

```tsx
import { Chart } from '@scaffald/ui';
```

## Examples

```tsx
// Line chart example
<Chart
  type="line"
  data={[
    { x: 'Jan', y: 10 },
    { x: 'Feb', y: 25 },
    { x: 'Mar', y: 18 },
    { x: 'Apr', y: 32 },
  ]}
  width={400}
  height={200}
/>

// Bar chart example
<Chart
  type="bar"
  data={[
    { label: 'Q1', value: 120 },
    { label: 'Q2', value: 180 },
    { label: 'Q3', value: 150 },
  ]}
/>
```
