# CryptoAssetCard

Card component displaying cryptocurrency asset information including price, change, and holdings.

## Import

```tsx
import { CryptoAssetCard } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <CryptoAssetCard
    name="Bitcoin"
    symbol="BTC"
    price="$43,250.00"
    change="+2.4%"
    changeDirection="up"
  />
</ThemeProvider>`} />

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/CryptoAssetCard/CryptoAssetCard.tsx) for the complete props API.
