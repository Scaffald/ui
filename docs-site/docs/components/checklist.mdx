# Checklist

Interactive checklist component for displaying tasks with completion states and a progress bar.

## Import

```tsx
import { Checklist } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Checklist
    title="Getting Started"
    subtitle="Complete these steps"
    completionPercentage={50}
    items={[
      { id: '1', title: 'Create your account', description: 'Sign up with email or SSO', complete: true },
      { id: '2', title: 'Verify your email', description: 'Check your inbox', complete: true },
      { id: '3', title: 'Set your preferences', description: 'Choose settings', complete: false },
      { id: '4', title: 'Invite team members', description: 'Add collaborators', complete: false },
    ]}
  />
</ThemeProvider>`} />

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | Header title |
| `subtitle` | `string` | — | Header subtitle |
| `items` | `ChecklistItem[]` | — | Array of checklist items |
| `completionPercentage` | `number` | — | Progress percentage (0–100) |
| `onItemPress` | `(item) => void` | — | Called when item is pressed |
| `isLoading` | `boolean` | `false` | Show skeleton loading state |
| `showProgress` | `boolean` | `true` | Show progress bar |

### ChecklistItem

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Unique identifier |
| `title` | `string` | Item title |
| `description` | `string` | Item description |
| `complete` | `boolean` | Whether completed |

## Loading State
