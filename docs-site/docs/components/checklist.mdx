# Checklist

Interactive checklist component for displaying a list of tasks or items with completion states.

## Import

```tsx
import { Checklist } from '@scaffald/ui';
```

## Basic Usage

<LivePreview code={`<ThemeProvider initialTheme="light">
  <Checklist
    items={[
      { id: '1', label: 'Set up your profile', completed: true },
      { id: '2', label: 'Connect your account', completed: true },
      { id: '3', label: 'Invite team members', completed: false },
      { id: '4', label: 'Complete onboarding', completed: false },
    ]}
  />
</ThemeProvider>`} />

## Props

See the [TypeScript definitions](https://github.com/Scaffald/ui/blob/main/src/components/Checklist/Checklist.tsx) for the complete props API.
