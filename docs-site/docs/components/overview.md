# Components Overview

Scaffald UI provides 90+ production-ready components organized into intuitive categories. All components are:

- ✅ **Cross-platform** - Work seamlessly on iOS, Android, and Web
- ✅ **Accessible** - WCAG 2.1 AA compliant with proper ARIA attributes
- ✅ **Type-safe** - Full TypeScript support
- ✅ **Themeable** - Support light/dark modes and custom themes
- ✅ **Tested** - Comprehensive test coverage

## Component Categories

### Form Components

Build beautiful, accessible forms with our comprehensive form component library:

- **[Button](./button)** - Primary, secondary, outline, ghost, and text variants
- **[TextInput](./text-input)** - Single-line text input with label and validation
- **[Textarea](./textarea)** - Multi-line text input
- **[Select](./select)** - Dropdown selection component
- **[Checkbox](./checkbox)** - Single or multi-selection checkboxes
- **[Radio](./radio)** - Mutually exclusive options
- **[Toggle](./toggle)** - Boolean on/off switch
- **[Slider](./slider)** - Range input component
- **[DatePicker](./date-picker)** - Date and time selection
- **[SearchBar](./search-bar)** - Search input with icon

### Layout

Structure your UI with flexible layout components:

- **[Stack](./stack)** - Vertical layout with spacing
- **[Row](./row)** - Horizontal layout with spacing
- **[Container](./container)** - Responsive container with max-width
- **[Divider](./divider)** - Visual separator
- **[Spacer](./spacer)** - Flexible spacing component
- **[Grid](./grid)** - Responsive grid layout

### Navigation

Navigate your app with these components:

- **[Tabs](./tabs)** - Tabbed navigation with multiple variants
- **[Breadcrumbs](./breadcrumbs)** - Hierarchical navigation
- **[Pagination](./pagination)** - Page navigation for lists
- **[Menu](./menu)** - Contextual menu
- **[Drawer](./drawer)** - Slide-out navigation panel

### Feedback

Provide user feedback with these components:

- **[Alert](./alert)** - Inline notification messages
- **[Toast](./toast)** - Temporary notification popups
- **[Modal](./modal)** - Modal dialog overlay
- **[Dialog](./dialog)** - Confirmation and alert dialogs
- **[Progress](./progress)** - Progress indicators
- **[Spinner](./spinner)** - Loading spinners
- **[Skeleton](./skeleton)** - Loading placeholders
- **[Badge](./badge)** - Status indicators and counts
- **[Tooltip](./tooltip)** - Contextual help text

### Data Display

Display data in beautiful, organized ways:

- **[Card](./card)** - Content container with variants
- **[Table](./table)** - Data tables with sorting and pagination
- **[List](./list)** - Vertical list of items
- **[Avatar](./avatar)** - User profile pictures
- **[Chip](./chip)** - Compact elements for tags and filters
- **[Icon](./icon)** - Icon component with 1000+ icons
- **[Image](./image)** - Optimized image component

### Typography

Text components for all your content needs:

- **[Text](./text)** - Base text component with variants
- **[Heading](./heading)** - Semantic heading levels (h1-h6)
- **[Link](./link)** - Interactive text links

### Overlays

Components that appear on top of content:

- **[Popover](./popover)** - Floating content container
- **[Dropdown](./dropdown)** - Dropdown menus and selects
- **[Sheet](./sheet)** - Bottom sheet and modal overlays

## Design Principles

All Scaffald UI components follow these core principles:

### Consistency
Every component uses the same design tokens, ensuring visual consistency across your entire application.

### Accessibility
Components include proper ARIA attributes, keyboard navigation, and screen reader support out of the box.

### Performance
Built with inline styles and optimized for React Native's rendering engine. Minimal runtime overhead.

### Flexibility
Extensive customization options through props while maintaining sensible defaults.
